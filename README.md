# 🎬 Catálogo Films

Um catálogo de filmes desenvolvido com **HTML, CSS e JavaScript**, consumindo a API do **The Movie Database (TMDB)** para exibir informações atualizadas sobre filmes populares e permitir pesquisas em tempo real.

## 📖 Sobre o projeto

O **Catálogo Films** foi criado com o objetivo de praticar conceitos fundamentais do desenvolvimento Front-End, como consumo de APIs, manipulação do DOM, armazenamento local e organização de código.

## ✨ Funcionalidades

* 🎥 Listagem de filmes populares
* 🔍 Pesquisa de filmes por nome
* 📄 Exibição de informações do filme
* ⭐ Nota de avaliação
* 📅 Ano de lançamento
* ❤️ Sistema de favoritos utilizando LocalStorage
* 🌙 Tema claro/escuro
* 📱 Layout responsivo

## 🚀 Como executar

1. Clone este repositório.

```bash
git clone https://github.com/seuusuario/catalogo-films.git
```

2. Entre na pasta do projeto.

```bash
cd catalogo-films
```

3. Crie um arquivo chamado `config.js`.

```javascript
const API_KEY = "SUA_CHAVE_DA_API";
```

4. Abra o arquivo `index.html` no navegador.
